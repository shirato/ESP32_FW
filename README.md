# ESP32_FW
ESP32 Restore Factory FW

##for Linux
###Firmware_Backup.sh
Backup ESP32 FW to esp32_firmware.bin 
###Firmware_Restore.sh
Restore ESP32 FW from esp32_firmware.bin
##for Windows
###Firmware_Backup.cmd
Backup ESP32 FW to esp32_firmware.bin 
###Firmware_Restore.cmd
Restore ESP32 FW from esp32_firmware.bin
